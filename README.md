# Trabajo: Gestor de Tareas Kanban

## Descripción general
Este proyecto corresponde a una evaluación de desarrollo web en la que se implementa un **gestor de tareas tipo Kanban** utilizando tecnologías básicas como HTML, CSS y JavaScript.

El objetivo es diseñar una aplicación funcional que permita organizar tareas en diferentes estados, aplicando conceptos de manipulación del DOM, eventos y estilos responsivos.

---

## Objetivo del trabajo
Desarrollar una aplicación web que permita:

- Crear tareas mediante un formulario
- Clasificar tareas en columnas Pendiente, En progreso, Completado
- Mover tareas entre columnas
- Eliminar tareas
- Mostrar contadores dinámicos por columna
- Aplicar diseño responsivo

---

## Tecnologías utilizadas

- HTML5 
- CSS3 
- JavaScript 
- Git y GitHub 

---

##  Funcionalidades implementadas

- Agregar tareas desde formulario
-  Organización en tablero Kanban
-  Movimiento de tareas entre columnas
-  Eliminación de tareas
-  Contadores automáticos por estado
-  Validación de campos obligatorios
-  Diseño adaptable a dispositivos móviles

---

## Conclusión

El desarrollo del proyecto permitió aplicar conocimientos teóricos en un caso práctico real. Además, se fortalecieron habilidades en programación web y resolución de problemas mediante la adaptación de código generado con ayuda de herramientas de inteligencia artificial.
